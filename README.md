## Description
Aplikasi sistem CRUD, USER dan PRODUK yang mana sistem memiliki 2 hak akses yaitu admin dan reseller. 

## Stack
- Reactjs
- Vitejs
- Tailwindcss

## Project setup
```bash
$ git clone https://github.com/rosfandy/geekgarden-frontend.git
```
```bash
$ npm install
```

## Run Program Development
```bash
$ npm run dev
```
